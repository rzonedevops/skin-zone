## Cheerleader Mascot Integration

- **Position**: Fixed in the bottom-right corner (desktop) or bottom-center (mobile)
- **Size**: Medium (128px on desktop, 64px on mobile)
- **Animation**: Standard cheering animation
- **Contextual Behavior**:
  - Shows supportive animation during salon selection
  - Displays excited animation when service is selected
  - Shows celebratory animation when booking is confirmed
  - Minimizes during form filling to reduce distraction
- **Interaction**:
  - Hovering reveals booking tips relevant to current step
  - Can be clicked to show personalized recommendations
  - Special animation plays when moving between steps

## AppDirect Integration

The booking flow integrates with AppDirect in the following ways:

1. **User Authentication**:
   - Single sign-on with AppDirect credentials
   - Access to user profile information

2. **Payment Processing**:
   - AppDirect Wallet payment option
   - Secure payment processing

3. **Subscription Management**:
   - Option to add booking to subscription services
   - Loyalty program integration

4. **Notification System**:
   - Email confirmations
   - Calendar integration
   - Reminder notifications

## HGNN Database Integration

The booking flow leverages HGNN database for:

1. **Personalized Recommendations**:
   - Suggested services based on user history
   - Therapist matching based on preferences

2. **Supply Chain Insights**:
   - Products used in selected treatments
   - Ingredient information for transparency

3. **Availability Optimization**:
   - Smart scheduling based on therapist expertise
   - Resource allocation efficiency

## Responsive Behavior

- Calendar view converts to date picker on mobile
- Time slot selection adapts to touch interface
- Form fields expand for easier input on mobile
- Mascot size reduces proportionally with screen size
- Step progress indicator simplifies on smaller screens
