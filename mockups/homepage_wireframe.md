## Cheerleader Mascot Integration

- **Position**: Fixed in the bottom-right corner (desktop) or bottom-center (mobile)
- **Size**: 200px (desktop), 96px (mobile)
- **Animation**: Continuously animated GIF
- **Behavior**: Remains visible at all times, even during scrolling
- **Interaction**: Hovering reveals minimize button and tenant-customizable message bubble
- **Accessibility**: Includes aria labels and respects reduced motion preferences

## Key UI Elements

1. **Navigation**:
   - Main categories: Ingredients, Products, Brands, Salons, Treatments, Therapists
   - User account access
   - Shopping cart
   - Search functionality

2. **Hero Banner**:
   - Tenant-customizable promotional content
   - Call-to-action buttons

3. **Product Showcases**:
   - Featured products with images, names, brands, and prices
   - Quick-view and add-to-cart options

4. **Salon Listings**:
   - Featured salons with images, names, locations, and ratings
   - Quick booking options

5. **Treatment Highlights**:
   - Trending treatments with images, names, durations, and prices
   - Salon availability indicators

6. **Supply Chain Insights**:
   - Interactive visualization powered by HGNN database
   - Shows connections between ingredients, products, brands, and salons
   - Highlights sustainability and ethical sourcing information

7. **Cheerleader Mascot**:
   - Persistent animated presence
   - Contextual animations based on user actions
   - Tenant-customizable appearance options
